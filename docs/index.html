Three targets, one simple physical gesture.

01

Handy

Uses Handy’s supported remote control to start, stop, and cancel transcription.

02

macOS Dictation

Sends the same double-modifier shortcut selected in your macOS Keyboard settings.

03

Custom shortcut

Define separate start and stop shortcuts for any app, then optionally send Enter.

Reversible by design.

Meeting mode stops the hardware helper completely. The PD200X firmware immediately owns the button again. The uninstaller removes the app, the login agent, and its preferences, while byte-for-byte guards prevent unsupported commands from reaching the microphone.
